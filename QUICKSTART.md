### Traditional Installation
```bash
# 1. Clone and install
git clone https://github.com/rapidrabbitsoft/vimconfig
cd vimconfig
./install.sh

# 2. Open Vim and install plugins
vim
:PlugInstall
:q

# 3. Done! Start coding
vim your-file.py
```
